📊 Calculadora de Média de Notas em JavaScript

Este repositório contém um exemplo simples de aplicação em JavaScript que calcula a média de um conjunto de notas e determina a situação final do aluno com base nesse resultado.

🚀 Funcionalidades

Cálculo da média aritmética de um array de notas

Verificação automática de aprovação ou reprovação

Exibição do resultado no console

📌 Regras

Média maior ou igual a 6 → Aluno aprovado

Média menor que 6 → Aluno reprovado

🧠 Conceitos utilizados

Arrays

Funções

Estruturas condicionais (if/else)

Operações matemáticas básicas

💻 Exemplo de uso

O código recebe uma lista de notas, calcula a média e imprime no console tanto a média quanto a situação do aluno.
